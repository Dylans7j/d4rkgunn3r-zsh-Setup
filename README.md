
# d4rkgunn3r Zsh Setup

A fast, clean, pentester-focused Zsh environment for Kali Linux.

This installer sets up:
- Oh My Zsh
- Powerlevel10k
- Nerd Fonts (Meslo)
- Zsh autosuggestions + syntax highlighting
- Optimized history settings
- Useful aliases for offensive security work

## Features
- One-command install
- Clean uninstall / reinstall friendly
- Designed for Kali, HTB, labs, and internal pentests
- No banners, no clutter, fast startup

## Installation

```bash
git clone https://github.com/YOURUSERNAME/d4rkgunn3r-zsh.git
cd d4rkgunn3r-zsh
chmod +x install.sh clean-install.sh 
./install.sh or clean-install.sh
